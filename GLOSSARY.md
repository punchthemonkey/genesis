# Glossary

| Term | Definition |
| :--- | :--- |
| **A2A** | Agent‑to‑Agent protocol for cross‑agent communication. |
| **Conductor** | The meta‑agent that composes task graphs and delegates to specialists. |
| **GOAT Standard** | Genesis engineering excellence framework. |
| **Hexagonal Architecture** | Ports & Adapters pattern; domain core isolated from infrastructure. |
| **IndexedDB** | Browser database for structured storage. |
| **MCP** | Model Context Protocol; standard for tool discovery. |
| **OPFS** | Origin Private File System; high‑performance file storage in browser. |
| **Orchestrator** | Domain service that executes the ReAct loop. |
| **PWA** | Progressive Web App; installable web application. |
| **ReAct** | Reasoning + Acting; LLM loop that interleaves thought and tool use. |
| **Result Monad** | A type representing either success (`Ok`) or failure (`Err`), forcing explicit error handling. |
| **Skill** | A packaged capability (system prompt + allowed tools) that can be installed. |
| **Turso** | Managed libSQL (SQLite fork) service with embedded replica sync. |
| **Vault** | Encrypted store for API keys and secrets. |
| **WebGPU** | Modern GPU API for the web, enabling fast on‑device ML inference. |
| **WebLLM** | Library that runs LLMs in the browser using WebGPU. |
